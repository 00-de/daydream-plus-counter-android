# 🌈 DayDreamプラス 題目カウンター DX+

リアルタイムで動作するシンプル＆高機能な題目カウンターアプリです📱✨

---

## 🚀 特徴

- ⏱️ リアルタイム題目カウント
- 📅 日・月・総数管理
- 🎯 目標達成率表示
- 🌈 達成演出（キャラメッセージ付き）
- 💾 ローカル保存（localStorage）
- 🙏 御祈念メモ機能
- 📱 スマホ完全対応（レスポンシブ）

---

## 📦 使い方

1. `index.html` を開く
2. 「▶ 自動開始」を押す
3. 題目をカウント開始
4. 目標達成で演出発動🎉

---

## 🌐 GitHub Pagesで公開する方法

1. GitHubでリポジトリ作成
2. `index.html` をアップロード
3. Settings → Pages を開く
4. Branch: `main`
5. Folder: `/root`
6. 公開URLが生成される

---

## 📱 Androidでも使用可能

WebViewアプリとしてそのまま利用できます。
（index.htmlをassetsに入れるだけでOK）
